![thumbnail](https://github.com/hmuawiyah/VOLTORA-Web-App-Power-Calculator/blob/main/screenshot/Thumbnail.png?raw=true)

VOLTORA is a web application designed to calculate estimated household electricity usage based on electricity tariffs set by PLN (the Indonesian state-owned electricity company).  
The calculations are based on estimated data and are not intended to be an official reference.





## Tech Stack
- React
- Next.js
- TypeScript
- Zustand
- Tailwind CSS
- shadcn/ui





## Features
- Light mode and dark mode
- Shareable estimation results via URL
- Download estimation results as an image
- Search feature to improve usability
